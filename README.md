# EduGuardIA Lang

Web app gamifiee de tutorat IA pour l'apprentissage de l'anglais (frontend vanilla + backend FastAPI).

```bash
# Installer les dependances Python
pip install -r requirements.txt
# Copier l'exemple d'environnement puis remplir HF_TOKEN
cp .env.example .env
# Lancer l'API et le frontend servis par FastAPI
uvicorn backend.main:app --reload
```
